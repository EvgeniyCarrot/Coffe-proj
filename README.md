# Реализация магазина для продажи вьетнамского кофе

## О проекте

Данный проект представляет собой одностраничный веб-сайт интернет-магазина, посвящённого продаже вьетнамского кофе. Сайт разрабатывается с использованием базовых веб-технологий: **HTML5**, **CSS3** и **JavaScript**.

### Этап 1: Семантическая вёрстка (HTML5)

На первом этапе была реализована **семантическая структура** сайта с использованием современных HTML5-элементов (`<header>`, `<main>`, `<section>`, `<nav>`, `<footer>` и др.).  

Основные работы на данном этапе:
- Создан HTML-каркас всех секций макета: главная, каталог продукции, подарочные наборы, контакты.
- Добавлена поддержка адаптивных изображений с использованием атрибутов `srcset` и формата **WebP** для улучшения скорости загрузки.
- Все изображения корректно подключены через относительные пути для корректной работы на GitHub Pages.
- Каждому элементу присвоены осмысленные классы и `id` для последующей стилизации с помощью CSS3.
- В код внесены поясняющие комментарии для упрощения дальнейшей разработки.
- Реализованы авторские улучшения структуры по сравнению с исходным макетом (например, семантическая группировка блоков, доступность и т.д.).
